# Путь к скриптам

Скрипты запуска и развёртывания находятся в каталоге **`scripts/`**. Запуск из корня проекта:

- `./scripts/deploy.sh` — развёртывание
- `./scripts/start_server.sh` — сервер
- `./scripts/start_bot.sh` — бот
- `./scripts/start_all.sh` — сервер и бот в фоне
- `./scripts/stop_all.sh` — остановка

Подробнее см. **`scripts/README.md`** в корне репозитория.

В документации в `docs/` в старых примерах может встречаться `./start_server.sh` и т.д. — замените на `./scripts/start_server.sh` и аналогично для остальных скриптов.
